How to Create Items

In order to create an item for listings, hover over "My Stuff" in the top right navigation bar and in the dropdown select "Items". On this page, you'll be able to see all the items you have created so far. In the top right below the navigation bar there will be a green button that says "Add Item". Clicking on this will take you the image creation form.

When creating an item, there are some requirements to keep in mind. First, an image must have at least one image selected in order to create the item. If you don't have any images uploaded to the site yet, you'll be able to upload images through the item creation form. An item must also have a name in order to create it. Descriptions are optional and not needed, but can be useful for other users to know what the item is. After creating your item, other users will not be able to see it until you add it to a listing.

How to Edit/Delete Items

In your list of items that you've created, under each image for an item, there will be two buttons, "Edit" and "Delete". Clicking on the Edit button will take you to the form to edit your item where you can change the name and description and add images to you item. Clicking on Delete button will take you to a confirmation page where you can confirm if you want to delete the item or not. Be warned though, if you delete an item, any listing that contains that item will also be deleted as well.

FAQs for Items

How many images can be used for an item?

You may use as many images as you want, but all of them must relate to the item you are creating. Items that contain images with no coherence will be deleted upon descretion.

How many items can I have?

You may have as many items as you need, but please delete items that you no longer use over time.

What can I use items for?

Items can be used for all kinds of listings (offers, auctions and search listings) as well as in wishlists. They can also be used in offers on other users listings.

Can I reuse items?

Yes, items can be used several times for each of the above instances. Items you have made will continue to exist even after the deletion of a listing, wishlist or offer.
